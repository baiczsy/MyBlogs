2.2. 代码相关¶

  • 普通代码定义：
::

    import sys
    from mod import main
    if __name__ == "__main__":
        sys.exit(main())

效果：

import sys
from mod import main
if __name__ == "__main__":
    sys.exit(main())
  • 高亮代码定义，带行号：
.. code-block:: python|console
    :linenos:

    import sys
    from mod import main
    if __name__ == "__main__":
        sys.exit(main())

效果:

1
2
3
4
import sys
from mod import main
if __name__ == "__main__":
    sys.exit(main())
  • 高亮代码定义2：

2.7. 列表¶

无序列表

- 首先
- 其次

  * 嵌套1
  * 嵌套2

- 再次

效果：

  • 首先
  • 其次
    • 嵌套1
    • 嵌套2
  • 再次

有序列表

#. 首先
#. 其次

   * 嵌套1
   * 嵌套2
#. 再次

效果：

  1. 首先
  2. 其次
    • 嵌套1
    • 嵌套2
  3. 再次
